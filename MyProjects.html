Countries App

SPA (Single Page Application).
Application where you can see information from 150 countries, brought from an external API and saved in a database. The data can be filtered and sorted. You can also create tourist activities. Technologies used: ReactJS, Redux, NodeJS, Express, PostgresSQL, Sequelize and CSS.

Wheather App

React app that brings meteorological data from different cities from an external api. A search bar is used to enter each city. Styles made with pure CSS.
